# Personal Finance Dashboard 🇭🇺

An interactive data analysis dashboard built to analyze 4 years of personal 
spending data exported from Revolut (2021-2024) during my studies in Budapest.

## What I Built
- Cleaned and categorized 2,000+ real transactions using Python and Pandas
- Built an interactive web dashboard using Plotly Dash
- Identified key spending patterns and insights from real personal data

## Key Findings
- Rent was the largest spending category
- I bought more clothes than I expected (even higher than travelling)
- In 2025, my expenses made 131% increase than 2024, due to inflation and unplanned purchases


## Tech Stack
- Python, Pandas — data cleaning and categorization
- Plotly, Dash — interactive visualization and dashboard
- Jupyter Notebook — exploratory data analysis
